# Карточки раскладов (превью)

Сюда кладите изображения превью раскладов. Имена файлов должны совпадать с теми, что использует приложение:

| Файл          | Расклад        |
|---------------|----------------|
| `basic.png`   | Быстрый (1 карта) |
| `quick.png`   | Базовый (3 карты) |
| `financial.png` | Финансы (5 карт) |
| `relationship.png` | Отношения (6 карт) |
| `celtic.png`  | Кельтский крест (10 карт) |

Для быстрой загрузки используйте изображения меньшего разрешения (например, 400–600 px по длинной стороне) и сжатие (WebP или JPEG 80–85%).
