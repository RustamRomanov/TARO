# Изображения колод таро

В каждой папке - карты одной колоды. Подхватывается автоматически из `frontend/src/data/tarotDecks.js`.

- **Имена папок** должны совпадать с путями в `tarotDecks.js`:  
  `Oriens Animal Tarot`, `The Dark Mansion Tarot`, `The Nightmare Before Christmas Tarot`, `Trionfi Della Luna`.
- **Рубашка колоды**: файл с «рубаш» в имени.
- **Превью колоды (витрина)**: файл с «колода» в имени.
- **Карты**: остальные файлы. Порядок - по числовому префиксу в имени файла.

Чтобы колоды и превью грузились быстрее, можно заменить изображения на версии с меньшим разрешением (например, 400–600 px по длинной стороне) и/или сжать (WebP, JPEG 80–85%). Имена файлов оставьте те же, положите в те же папки.
