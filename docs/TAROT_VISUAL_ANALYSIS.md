# Визуальный анализ карт Таро

**Актуализация структуры проекта:** апрель 2026. Тематический документ; общая архитектура: [README.md](./README.md).

Чтобы ИИ использовал описание изображения каждой карты при толковании, нужно один раз выполнить скрипт анализа.

## Шаг 1. Запуск скрипта

```bash
cd ASTROV
source venv/bin/activate
python scripts/analyze_tarot_cards.py
```

Требуется:
- `AI_API_KEY`, `AI_BASE_URL`, `AI_VISION_MODEL` в `.env`
- Колоды в `frontend/src/assets/taro/`

Скрипт:
1. Сканирует колоды, указанные в `FOLDER_TO_DECK_ID` (включая Rider-Waite Tarot)
2. Отправляет каждую картинку в Vision API с промптом «Опиши детально эту карту Таро»
3. Сохраняет описания в `app/data/tarot_card_descriptions.json` — структура `{ "deck_id": { "card_id": "описание" } }`

## Шаг 2. Использование

После успешного запуска приложение автоматически подставляет описание карты в промпт при толковании. Фронтенд передаёт `deck`, `card_id` (например, `egyptian`, `Disks04`), бэкенд ищет описание в JSON и добавляет его в запрос к ИИ.
